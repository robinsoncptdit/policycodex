The policy policies/{{ policy.slug }} is marked foundational: it supplies configuration values that other policies and the app itself depend on (for example the document classification list and retention schedule).

To change a foundational policy, use the typed-table editor in the onboarding wizard or the foundational-policy admin screen. The edit form on this page is only for ordinary policy documents.
