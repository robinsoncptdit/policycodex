SUPERSEDED 2026-06-11

This storyboard described the pre-pivot architecture.

The earlier version of this page walked through a seven-screen onboarding flow. The live DISC-readiness walkthrough on 2026-06-11 surfaced architectural fault lines in that flow, and the team paused the demo to redesign.

The replacement architecture is a GitHub-style /settings/ page with five panels plus a top-level /inventory/ page. Configuration moves into a settings surface admins can return to at any time.

A fresh storyboard for the Settings architecture will land here once the rebuild ships. In the meantime, the public README and the ROADMAP carry the current status.
