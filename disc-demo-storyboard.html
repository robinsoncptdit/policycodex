REPO-05

The IT director clones the public repo and runs one command. Docker Compose handles everything: builds the image, runs migrations, starts gunicorn, persists state to a named volume.

A note on this step at v0.1

v0.1 ships local-folder ingest. The drop-files visual surface above and the cloud connectors (SharePoint, OneDrive, Google Drive, Box, Dropbox) are v0.2 work, prioritized by what adopters actually ask for. Publisher merges, handbook builds automatically
minute 28 to 30
APP-19 + PUBLISH-07

An admin with merge permission clicks Publish, which merges the PR. The merge triggers the GitHub Actions handbook build. Astro generates static HTML. GitHub Pages deploys it to the custom subdomain. Within five minutes, the change is live.

GitHub Actions: Build handbook Run pnpm install Run pnpm build · Astro generated 187 pages Run actions/upload-pages-artifact Run actions/deploy-pages ✓ Deployed to https://handbook.ptdiocese.org/
